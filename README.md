# 자투리 (Jaturi)
AI와의 대화를 통해 일기와 독후감을 자동 생성하는 기록 서비스

Status: v1.0 MVP 개발 완료
- 개발 기간: 2025.10 ~ 2025.12
- MVP 기준 기능 구현 완료
- 현재 리팩토링 및 구조 개선을 계획 중

---

## 📌 프로젝트 개요

자투리는 글을 쓰고 싶지만  
어디서부터 어떻게 시작해야 할지 막막한 사용자를 위해  
AI 질의응답 기반으로 생각을 수집하고  
이를 구조화된 일기·독후감으로 완성해주는 서비스입니다.

MVP 단계에서는 글쓰기 실력 향상이 아닌  
**글쓰기에 대한 부담을 줄이는 것**에 초점을 맞췄습니다.

## ✨ 핵심 기능

- AI 질의응답 기반 일기 작성
- AI 질의응답 기반 독후감 작성
- KoBERT 기반 감정 분석 (7가지 감정 분류)
- 감정 기반 음악 / 도서 추천
- 사용자 만족도 피드백 반영 재생성

---

## 🔧 부가 기능

- 회원가입 / 로그인 (JWT, SNS 간편 로그인)
- 친구 관리 및 친구 글 조회
- 자투리 기록 리스트
- 마이페이지 (프로필 수정)
- 관리자 페이지 (회원 / 통계 관리)

---

## 🔄 서비스 플로우

1. 일기 / 독후감 유형 선택  
2. AI 맞춤 질문 응답 (약 5회)  
3. 감정 분석  
4. 글 통합 및 생성  
5. 콘텐츠 추천 (음악 / 도서)  
6. 사용자 피드백 수집  

---

## 🛠 기술 스택

### Frontend
- React (웹)
- React Native (앱, MVP)

### Backend
- Java 17
- Spring Boot
- Spring Security
- JWT 기반 인증 / 인가

### AI
- PyTorch
- KoBERT (감정 분석)
- GPT API (질문 생성 / 글 통합)

### Database
- MariaDB

---

## 🏗 시스템 아키텍처

- Frontend ↔ Backend : REST API 통신  
- Backend ↔ AI Server : 감정 분석, 질문 생성 및 글 통합 요청
- Backend ↔ Database : 사용자 및 콘텐츠 데이터 관리
